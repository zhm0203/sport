# 校园动力 (CampusFit) - 产品需求文档

## 1. 产品概述

**产品名称**: 校园动力 (CampusFit)  
**产品类型**: 游戏化运动激励小程序  
**目标用户**: 大学生群体（18-25岁）  
**核心价值主张**: 通过"即时反馈 + 社交竞争 + AI陪伴"，打造大学生专属的运动成长乐园

### 1.1 产品背景
当代大学生普遍存在运动量不足、运动坚持率低的问题。传统的运动记录应用缺乏吸引力和激励性，难以形成长期习惯。本产品通过游戏化设计、AI智能陪伴和社交互动，革新大学生运动体验。

### 1.2 目标与关键指标
- **DAU 目标**: 上线6个月达到日活 50万+
- **运动坚持率**: 提升用户周运动3次以上的比例至 60%
- **留存率**: 30日留存率目标 45%
- **社交渗透率**: 30%用户形成运动搭子关系

## 2. 用户画像

### 2.1 核心用户画像
**小美 - 大一新生**
- 背景：刚入学，希望认识新朋友，融入校园生活
- 痛点：不认识人，缺乏运动动力，想改变但难以坚持
- 需求：轻松的社交方式、趣味性的运动激励

**阿杰 - 大三学生**
- 背景：课业繁忙，渴望运动但时间碎片化
- 痛点：运动计划总被学习打断，缺乏即时激励
- 需求：灵活的运动记录、碎片化激励机制

**小林 - 研究生**
- 背景：科研压力大，需要释放压力、保持健康
- 痛点：运动枯燥乏味，缺乏成就感
- 需求：深度成就系统、可视化的成长路径

## 3. 核心功能模块

### 3.1 运动数据记录系统
**功能描述**: 自动化记录用户运动行为，实时展示运动成果

**功能列表**:
1. **步数追踪**
   - 实时步数展示（当前/目标/百分比）
   - 步数曲线图（今日/本周/本月）
   - 步数预测（基于历史数据）

2. **卡路里计算**
   - 运动消耗卡路里（走路、跑步、骑行）
   - 今日摄入/消耗对比
   - 周卡路里趋势

3. **打卡系统**
   - 运动打卡（支持拍照上传）
   - 连续打卡天数展示
   - 打卡成就解锁

4. **运动类型支持**
   - 走路（自动识别）
   - 跑步（GPS轨迹记录）
   - 骑行（GPS轨迹记录）
   - 其他运动（自定义）

### 3.2 游戏化成长系统
**功能描述**: 通过等级、EXP、徽章等元素，提供即时成就反馈

**功能列表**:
1. **等级系统**
   - 等级范围：Lv.1 - Lv.100
   - 等级称号（运动小白 → 运动达人 → 校园传奇）
   - 升级动画和特效

2. **经验值(EXP)系统**
   - 每日签到：+10 EXP
   - 步数达成：每1000步 +5 EXP
   - 打卡：+20 EXP
   - 连续打卡：额外 +10 EXP/天
   - 社交互动：+5 EXP/次

3. **徽章系统**
   - 初始徽章：新手启程、初次打卡
   - 累计徽章：百日达人、千里之行
   - 社交徽章：社交蝴蝶、运动搭子
   - 稀有徽章：限时活动、特殊成就

4. **成就墙**
   - 已解锁成就展示
   - 进度可视化
   - 下一个目标预览

### 3.3 校园地图探索
**功能描述**: 虚拟校园地图探索，激励用户解锁校园地标

**功能列表**:
1. **虚拟地图**
   - 校园全景地图（2D俯视图）
   - 建筑地标（图书馆、体育馆、食堂等）
   - 解锁状态可视化

2. **地标解锁**
   - 步数累计解锁（每10000步解锁一个地标）
   - 打卡解锁特定地标
   - 隐藏地标（特殊成就触发）

3. **地图收集**
   - 地标卡片收集
   - 地标故事介绍
   - 校园知识科普

### 3.4 社交匹配系统
**功能描述**: 基于兴趣和运动偏好，智能匹配运动搭子

**功能列表**:
1. **兴趣匹配**
   - 运动类型偏好
   - 运动时间偏好
   - 运动目标设定

2. **组队功能**
   - 发起组队邀请
   - 接受/拒绝组队请求
   - 队伍信息展示

3. **运动搭子**
   - 搭子匹配列表
   - 搭子主页信息
   - 搭子运动数据同步

4. **排行榜**
   - 步数排行榜（全校/学院/班级）
   - 连续打卡排行榜
   - 周/月/总榜

### 3.5 AI 运动搭子
**功能描述**: 智能AI助手，提供个性化运动建议和情感支持

**功能列表**:
1. **智能对话**
   - 日常问候与关心
   - 运动提醒
   - 疑问解答

2. **个性化建议**
   - 运动计划推荐
   - 运动强度调整
   - 恢复建议

3. **激励系统**
   - 适时鼓励
   - 成就祝贺
   - 连续坚持提醒

4. **情绪识别**
   - 感知用户情绪
   - 提供情感支持
   - 调整激励策略

### 3.6 任务与奖励系统
**功能描述**: 丰富的任务体系和奖励兑换，提升用户粘性

**功能列表**:
1. **每日任务**
   - 每日步数目标
   - 每日打卡
   - 社交互动任务
   - EXP双倍卡任务

2. **每周挑战**
   - 周累计步数挑战
   - 周打卡连续挑战
   - 社交组队挑战

3. **奖励兑换**
   - 运动装备兑换
   - 校园服务兑换（场馆预约优先）
   - 虚拟物品兑换（头像框、表情包）
   - 商家优惠券

## 4. UI/UX 设计规范

### 4.1 视觉风格
**设计定位**: 深色科技感 + 轻游戏化 + 年轻化

**配色方案**:
- 主色调：#6366F1（靛蓝紫，代表活力与科技）
- 次要色：#10B981（翠绿，代表健康与成长）
- 强调色：#F59E0B（琥珀色，代表成就与奖励）
- 背景色：#0F172A（深空蓝黑）
- 卡片背景：#1E293B（深灰蓝）
- 文字主色：#F8FAFC（亮白）
- 文字次色：#94A3B8（银灰）

**字体规范**:
- 主标题：Unbounded（未来感强）
- 正文：Noto Sans SC（清晰易读）
- 数字强调：Orbitron（科技感）

**圆角系统**:
- 小元素：8px
- 卡片：16px
- 大卡片：24px
- 按钮：12px

**间距系统**:
- 基础单位：4px
- 小间距：8px
- 中间距：16px
- 大间距：24px
- 区块间距：32px

### 4.2 组件规范
**按钮样式**:
- 主按钮：渐变背景（主色→次色），白色文字
- 次按钮：边框样式，主色边框
- 幽灵按钮：透明背景，主色文字

**卡片样式**:
- 毛玻璃效果（backdrop-filter: blur）
- 微光边框（subtle glow）
- 轻微阴影

**动画效果**:
- 页面切换：fade + slide，300ms
- 按钮点击：scale(0.95)，150ms
- 数据更新：countUp 动画
- 徽章解锁：burst 特效
- 升级动画：光效 + 粒子效果

### 4.3 响应式策略
**移动优先**: 所有组件基于375px宽度设计
**适配范围**: 320px - 428px（主流手机尺寸）
**布局策略**: 单栏布局，核心信息优先展示
